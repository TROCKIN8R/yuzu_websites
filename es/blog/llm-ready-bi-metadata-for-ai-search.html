Metadatos BI listos para LLM: modelos semánticos que responden bien a búsqueda IA

Por Adrien Yvin · jul. 22, 2026

aillmmetadatapower-biseo

Motores y LLM no leen su DAX. Leen nombres, descripciones y docs. Metadatos vacíos → definiciones inventadas.

Qué consumen los sistemas IA

Descripciones de tablas/columnas, nombres de medidas, títulos de informes, docs externos (llms.txt, wiki).

Trate metadatos como documentación API pública.

Estándar mínimo

Por cada modelo certificado: grano en descripción de tabla, significado de columna, intent de medida en lenguaje claro, línea de sinónimos.

llms.txt y SEO IA

Alinee vocabulario docs ↔ modelo. Publique página texto con datasets certificados, owners y glosario.

Bloquee promote a Prod si medidas clave sin descripción.

Reserve una llamada de 30 minutos para un sprint de metadatos.
