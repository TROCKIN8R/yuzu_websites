Moteurs et LLM ne lisent pas votre DAX. Ils lisent **noms, descriptions et docs**. Métadonnées vides → définitions inventées.

## Ce que consomment les systèmes IA

Descriptions tables/colonnes, noms de mesures, titres de rapports, docs externes (llms.txt, wiki).

Traitez les métadonnées comme une **doc API publique**.

## Standard minimal

Pour chaque modèle certifié : grain en description table, sens métier colonne, intent mesure en langage clair, ligne de synonymes.

## llms.txt et SEO IA

Alignez le vocabulaire docs ↔ modèle. Publiez une page texte listant datasets certifiés, owners et glossaire.

Bloquez promote Prod si mesures clés sans description.
